# 業務レベル要件定義

## 業務レベルの評価軸

### 影響範囲による評価
- **組織**: 所属組織内 / 部門横断 / 全社的 / グループ会社含む
- **地域**: 国内のみ / 国内中心 / 国内外含む / グローバル統括

### 難易度による評価
- **業務複雑性**: 定型業務 / 半定型業務 / 非定型業務 / 創造的業務
- **意思決定レベル**: 実行レベル / 戦術レベル / 戦略レベル / 経営レベル
- **専門性要求度**: 基礎知識 / 応用知識 / 専門知識 / エキスパート知識

<br>

## レベル定義

| レベル | 組織 | 地域 | 業務複雑性 | 意思決定レベル | 専門性要求度 | 特徴 |
|--------|------|------|-----------|---------------|-------------|------|
| **ジュニアレベル** | 所属組織内 | 国内のみ | 定型業務 | 実行レベル | 基礎知識 | 既定の手順・基準に従った業務実行、上位者の指導下での遂行 |
| **中級レベル** | 所属組織内〜部門横断 | 国内のみ〜国内中心 | 半定型業務 | 実行レベル〜戦術レベル | 基礎知識〜応用知識 | 一定の裁量権を持った業務遂行、チーム指導、改善提案 |
| **エキスパートレベル** | 部門横断〜全社的 | 国内中心〜国内外含む | 非定型業務 | 戦術レベル〜戦略レベル | 専門知識 | 専門領域でのリーダーシップ、制度設計・仕組み構築、重要案件担当 |
| **シニアレベル** | 全社的〜グループ会社含む | 国内外含む〜グローバル統括 | 創造的業務 | 戦略レベル〜経営レベル | エキスパート知識 | 全社・グループレベルでの戦略策定・統括、経営層への直接報告・提案 |
